# Component Screenshots

Place screenshots in this directory with the following filenames:

## SimPlaylist
- `simplaylist-overview.png` - Main view with album grouping and tracks
- `simplaylist-settings.png` - Preferences page with grouping patterns

## Playlist Organizer
- `plorg-overview.png` - Main tree view with folders and playlists
- `plorg-import-menu.png` - Import menu dropdown
- `plorg-path-mapping.png` - Path mapping dialog
- `plorg-settings.png` - Preferences page

## Waveform Seekbar
- `waveform-overview.png` - Seekbar with waveform visualization
- `waveform-settings.png` - Preferences page with color/style options

## Last.fm Scrobbler
- `scrobbler-settings.png` - Preferences page with account and options

## Guidelines
- PNG format preferred
- Reasonable dimensions (800-1200px width typical)
- Dark theme screenshots preferred for consistency
- Crop to show relevant UI elements
